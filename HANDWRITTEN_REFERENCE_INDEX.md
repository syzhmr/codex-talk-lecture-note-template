# 手書きノート由来の参考文献候補

このファイルは，手書きノートから拾った参考文献候補を記録する台帳である。
講義中の主張と参考文献の対応を探す前に，まずこのファイルを更新する。

| ID | ノート箇所 | 表記 | 種類 | 判読状況 | 候補文献 | 照合状況 | 関連主張 | 備考 |

## 種類

- URL
- 著者名
- 書名・論文名
- 講義ノート名
- arXiv 番号
- DOI
- Stacks tag
- MathOverflow / Math StackExchange 番号
- 定理番号
- 命題番号
- 補題番号
- 節番号
- ページ番号
- 文献略記
- 色付き矢印
- 引き出し線
- 丸囲み・下線

## 判読状況

| 状態 | 意味 |
| --- | --- |
| 判読済み | 文字列として読める。 |
| 一部判読 | 文献名や番号の一部だけ読める。 |
| 要画像確認 | 元画像または高解像度版の確認が必要。 |
| 未判読 | 参考候補らしいが読めない。 |

## 照合状況

| 状態 | 意味 |
| --- | --- |
| 未照合 | まだスライド，配布資料，参考文献台帳と照合していない。 |
| 候補あり | 候補文献は見つかったが，該当箇所未確認。 |
| 箇所確認中 | 定理番号，節番号，ページ，URL を確認中。 |
| 確定 | 対応箇所を確認済み。 |
| 不採用 | 講義中の主張とは対応しない。 |
| 要確認 | 判読不能または資料間衝突がある。 |

## 運用

- 手書きノートにある参考文献候補を無視しない。
- URL や文献名が文章として書かれていなくても，色付き矢印，引き出し線，丸囲み，下線が根拠候補を指している場合は記録する。
- 判読できない候補は推測で本文に入れず，未判読として残す。
- 候補が `CLAIM_REFERENCE_MAP.md` の主張に関係する場合は，関連主張欄に主張 ID を書く。
