# 参考文献メモ

このディレクトリには，参考文献ごとの確認メモを置く。
講義中の主張と文献の対応を確認した場合は，文献名，該当箇所，講義中の主張との関係を記録する。

## 探す順序

1. `HANDWRITTEN_REFERENCE_INDEX.md` で手書きノート由来の候補を確認する。
2. `CLAIM_REFERENCE_MAP.md` で講義中の主張との対応状態を確認する。
3. `CODEX_REFERENCE_CANDIDATES.md` で Codex 提案候補かどうかを確認する。
4. `references/REFERENCE_INDEX.md` で文献の所在と確認状況を確認する。
5. このディレクトリの個別メモを確認する。
6. 必要な PDF または公開 URL を確認する。

## 個別メモの推奨形式

```md
# [文献名]

## 書誌情報

- 著者:
- タイトル:
- 出版情報:
- URL / DOI:
- 最終確認日:

## 講義中の対応

| 講義箇所 | 主張ID | 文献箇所 | 一致度 | 注意 |
| --- | --- | --- | --- | --- |
| Talk 1 / T1-01 | CL-001 | Theorem [ ], p.[ ] | 完全一致 |  |

## 引用種別

- 主引用: 講義中の主張に直接対応する引用。手書きノートまたは配布資料由来の文献を優先する。
- 補足引用: 一般的に飛ばされている背景，標準事実，記号確認のための補助引用。本文の主線を置き換えない。
- 台帳のみ: 本文には出さず，対応候補として記録する。

## 記号差分

| 講義中 | 文献中 | 注意 |
| --- | --- | --- |
| [記号] | [記号] |  |

## 未確認

- [未確認点]
```

## 注意

- 参考文献の証明を本文へ補完しない。
- 講義中の主張と仮定が一致しない場合は，その差分を記録する。
- Codex 提案参考文献は，講義中に実際に参照された文献として扱わない。
- 補足引用を，講義中の主張に直接対応する主引用として扱わない。
- 未確認のページ番号や定理番号を推測で書かない。
