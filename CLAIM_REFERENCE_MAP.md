# 講義中の主張と参考文献の対応表

## 文献出自

| 出自 | 意味 |
| --- | --- |
| 手書き由来 | 手書きノートに文献名，URL，略記，定理番号などが記録されている。 |
| スライド由来 | スライドの参考文献欄，脚注，URL に載っている。 |
| 配布資料由来 | 配布資料の本文，注，参考文献欄に載っている。 |
| 板書由来 | 板書写真に文献名，URL，定理名などが写っている。 |
| Codex 提案 | 講義資料の外から Codex が提案した候補。講義中に実際に参照された文献として扱わない。 |

## 引用種別

| 種別 | 意味 |
| --- | --- |
| 主引用 | 講義中の主張に直接対応する引用。手書きノートまたは配布資料由来の文献を優先する。 |
| 補足引用 | 一般的に飛ばされている背景，標準事実，記号確認のための補助引用。本文の主線を置き換えない。 |
| 台帳のみ | 本文には出さず，対応候補として記録する。 |

## 一致度

| 一致度 | 意味 |
| --- | --- |
| 完全一致 | 仮定，対象，結論が一致する。 |
| 記号違いだが内容一致 | 参考文献側の記号は異なるが，内容は一致する。 |
| 仮定が一部違う | 結論は近いが仮定が講義中の主張と異なる。 |
| 結論が類似 | 関連するが，同じ主張とは言えない。 |
| 関連箇所だが対応未確定 | 文献内に関連箇所はあるが，対応が確定していない。 |
| 未発見 | 対応箇所がまだ見つかっていない。 |

## 状態

| 状態 | 意味 |
| --- | --- |
| 未確認 | 主張または文献候補をまだ確認していない。 |
| 照合中 | 文献の該当箇所を確認中。 |
| 確定 | 対応箇所を確認し，本文引用に使える。 |
| 要確認 | 仮定，記号，結論，ページ番号などに未確認点がある。 |
| 未発見 | 対応文献が見つかっていない。 |
| 不採用 | 候補文献は講義中の主張に対応しない。 |

## 本文での扱い

| 扱い | 意味 |
| --- | --- |
| 本文引用 | 本文または脚注で引用する。 |
| 台帳のみ | 本文には出さず，作業台帳に残す。 |
| 未反映 | 未確認のため本文へ反映しない。 |
| 要ユーザー確認 | ユーザー判断が必要。 |

## 運用

- 講義中の主張は，手書きノートを入口に抽出する。
- 板書，スライド，配布資料で主張の形を確認する。
- 参考文献対応は，文献名だけでなく定理番号，命題番号，節番号，ページ，URL まで確認する。
- 仮定が違う場合は，完全一致としない。
- 証明補完のために参考文献を使わない。
- 対応が確定した場合でも，講義中に記録されていない証明を本文へ追加しない。
