# 未解決点・資料差分

このファイルは，判読不能，資料不足，引用未特定，資料間衝突，講演者確認が必要な点を記録する台帳である。
未解決点を本文へ曖昧に混ぜないために使う。

## 種類

- 判読不能
- 引用未特定
- 定理番号未確認
- URL 未確認
- 記号衝突
- 資料間衝突
- 講演者確認が必要
- 証明補完禁止
- 外部検索が必要

## 状態

| 状態 | 意味 |
| --- | --- |
| 未確認 | まだ確認していない。 |
| 確認中 | 資料または文献を確認中。 |
| 要ユーザー確認 | ユーザー判断が必要。 |
| 解消 | 確認が終わり，本文または台帳に反映済み。 |
| 保留 | 現時点では解決できない。 |

## 運用

- 判読できない文献名，URL，定理番号を推測で本文へ入れない。
- 講義中の主張と参考文献の仮定が一致しない場合は，本文へ混ぜずここに記録する。
- 証明を補いたくなったが，講義中の記録にない場合は「証明補完禁止」として記録する。
- 解消した項目は，本文または該当台帳に反映した後で削除してよい。
