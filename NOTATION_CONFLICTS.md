## 状態

| 状態 | 意味 |
| --- | --- |
| 未確認 | 衝突を記録しただけで，採用記法未決定。 |
| 確認中 | 資料を照合中。 |
| 採用済み | 本文で使う記法を決定済み。 |
| 要ユーザー確認 | ユーザー判断が必要。 |
| 解消 | 衝突ではないと確認済み。 |

## 運用

- 原則として講義中の記法を優先する。
- 手書きノートの記号が不明確な場合は，板書，スライド，配布資料で確認する。
- 参考文献側の記号へ本文を寄せる場合は，理由を記録する。
- 同じ文字が別対象に使われている場合は，本文で混乱しないよう注記または記法変更を検討する。
- 記法変更は本文へ反映する前に，講義記録としての忠実性を損なわないか確認する。
