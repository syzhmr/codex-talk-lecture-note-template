# 集中講義提示参考文献の活用監査

この台帳は，集中講義中に提示された参考文献を，本文，主張対応，記号確認，日本語表現確認，未確認点のどこで使ったかを記録するための監査表である。
参考文献は，証明補完のためではなく，講義中の主張の対応箇所確認，記号確認，表現確認，書誌情報確認のために使う。

## 運用

- 講義中に挙げられた参考文献は，`references/REFERENCE_INDEX.md` への登録だけで完了扱いにしない。
- どの手書きページ，どの主張，どの記号，どの本文箇所の確認に使ったかを記録する。
- スキャン PDF または検索不能 PDF は画像化して確認し，画像ファイル名，印刷ページ，節番号，定理番号，命題番号を残す。
- 対応主張が未発見または未確定の場合も，未確認点として残す。
- 参考文献側の記号と講義中の記号が異なる場合は，`NOTATION_CONFLICTS.md` にも記録する。
- 参考文献側の主張で講義中に記録されていない証明を補わない。

## 監査表

| ID | 講義中の出典 | 参考文献 | 所在 | 画像確認・確認箇所 | 使用箇所 | 未確認点 |
| --- | --- | --- | --- | --- | --- | --- |
| LRU-001 | 手書き p. |  | `references/...` |  |  |  |

## 使用箇所の分類

- 本文引用: 完成 PDF 本文の引用として使う。
- 主張対応: `CLAIM_REFERENCE_MAP.md` の対応確認に使う。
- 記号確認: 講義中の記法と参考文献側の記法の差分確認に使う。
- 表現確認: 日本語表現，用語，定理名の確認に使う。
- 書誌確認: URL，DOI，出版情報，版情報の確認に使う。
- 未確認点: 該当箇所が未発見または対応未確定。
