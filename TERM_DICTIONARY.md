# 用語辞書

このファイルは，完成版ノートで採用する用語，避ける表現，ユーザー確認前の候補を管理する。
外側テンプレートや他プロジェクトの用語辞書はそのまま流用せず，対象講義の手書きノートと参考資料に基づいて更新する。
このテンプレートは配布単位で自己完結する。外側の親ディレクトリや別リポジトリに共通辞書があることを前提にしない。

## 運用

1. 手書きノートの表現を最優先する。
2. 手書きノートで英語表記の語は，必要がない限り英語のまま残す。
3. 日本語化する場合は，手書きノート内の日本語表記，講演者配布資料，日本語参考文献の順に確認する。
4. 日本語参考資料は表現確認用であり，数学的主張の本文追加や証明補完の根拠にはしない。数学的引用として使う場合は，別途 `WORK_LEDGER.md` に対応を記録する。
5. 未確認の訳語は本文へ直接反映せず，このファイルの候補欄に残す。
6. 採用，不採用，保留の判断が確定した語だけを採用欄または使用しない欄に移す。
7. 手書きノートで判読できる日本語表現は，日本語参考文献の標準表現より優先する。
8. 数学的に自然な補いを用語選択の根拠にしない。手書きノートにない整理語，分類語，接続語は本文へ入れず，必要なら `WORK_LEDGER.md` に残す。
9. 複数プロジェクトで使える標準語でも，手書きノートや講演者資料が別表現を明示している場合は，対象講義の表現を優先し，上書き理由をこの辞書に残す。

## 共通化する基準

- 複数の講義ノートで同じ表現を使っても衝突しにくい標準的な数学用語は，この辞書の採用欄に初期値として置いてよい。
- 人名表記，ハイフン，TeX 表記など，全体で揃える価値がある表記は，この辞書で管理する。
- 手書きノート，講演者資料，配布資料に依存する語は，共通語として扱わず，対象講義固有の上書きとして管理する。
- 未承認語，共通化できるか不明な語，プロジェクト間で衝突しそうな語は，本文へ直接入れず候補欄に置く。

## 共通化しないもの

| 表現 | 理由 |
|---|---|
| 手書きノートで英語維持された専門語 | ノート作成者・講演者の表現選択を優先するため，共通の日本語訳を固定しない。 |
| `loc`，`res`，`Inf`，`Res` などの写像名 | 手書きノート・講演者資料・参考文献に実際に出ている場合だけ導入する。 |
| まず / 次に / 最後に などの接続語 | 講義ごとの文体方針に依存するため，共通辞書では固定しない。 |
| 参考文献の一時的な言い換え | 本文で採用する前に，対象講義の記号・文体に戻して確認する必要がある。 |
