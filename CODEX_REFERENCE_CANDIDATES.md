# Codex 提案参考文献候補

このファイルは，Codex が講義資料の外から提案した参考文献候補を記録する台帳である。
講義中に参照された文献，手書きノートに書かれた文献，スライド・配布資料に載っている文献とは出自を分けて管理する。

## 基本方針

- Codex 提案参考文献は，講義由来の参考文献候補を置き換えない。
- Codex 提案参考文献は，引用未特定，記号衝突，主張の標準的出典確認，書誌情報確認，一般的に飛ばされている背景の補足引用候補として扱う。
- 講義中に記録されていない証明，中間補題，背景定理を本文へ追加するためには使わない。
- 採用前に，なぜその文献が必要になったかを記録する。
- 採用前に，講義中の主張との関係を `CLAIM_REFERENCE_MAP.md` に記録する。

## 提案理由

- 引用未特定: 講義中の主張に対応する文献候補が手書きノートやスライドから特定できない。
- 記号衝突: 講義中の記号と参考文献側の記号の関係を確認したい。
- 標準出典確認: 講義中の主張が標準的な定理名・命題名を持つか確認したい。
- 書誌情報確認: 講義中に挙がった文献の正確な書誌情報を確認したい。
- URL 補完: 手書きノートの URL が判読不能で，候補 URL を確認したい。
- 補足引用: 一般的に飛ばされている背景，標準事実，記号確認のための補助引用候補を確認したい。

## 状態

| 状態 | 意味 |
| --- | --- |
| 未確認 | 候補として挙げただけで，文献内容を確認していない。 |
| 確認中 | 該当箇所，定理番号，ページ，URL を確認中。 |
| 候補採用 | 講義中の主張との関係が確認でき，台帳に採用する。 |
| 台帳のみ | 本文には出さず，補助候補として残す。 |
| 不採用 | 講義中の主張や記号確認には対応しない。 |
| 要ユーザー確認 | 採用するかユーザー判断が必要。 |

## 採用条件

Codex 提案参考文献を本文または `CLAIM_REFERENCE_MAP.md` で採用するには，次を満たす。

1. 手書きノート，板書，スライド，配布資料，既存参考文献台帳を先に確認している。
2. 提案理由が明確である。
3. 文献の書誌情報，URL，該当箇所，定理番号，命題番号，ページのいずれかを確認している。
4. 講義中の主張との関係を，完全一致，記号違いだが内容一致，仮定が一部違う，結論が類似，関連箇所だが対応未確定のいずれかに分類している。
5. 主引用として使うのか，補足引用に留めるのかを分類している。
6. 記号が違う場合は `NOTATION_CONFLICTS.md` に記録している。
7. 証明補完のためではないことを確認している。

## 禁止事項

- Codex 提案参考文献を，講義中に実際に参照された文献として扱わない。
- 手書きノート由来の未判読候補を，Codex 提案文献で勝手に置き換えない。
- 対応箇所が見つかっただけで，講義中の主張と同一視しない。
- 証明補完のために本文へ追加しない。
- 補足引用を，講義中の主張に直接対応する主引用として扱わない。
- 定理番号，ページ番号，URL を推測で書かない。
